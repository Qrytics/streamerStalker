Features

🔴

Twitch Live Alerts

Polls the Twitch Helix API every 60 seconds and sends a rich embed the moment a streamer goes live — with title, game, viewer count, and stream thumbnail.

🐦

Twitter / X Updates

Fetches new tweets via Nitter RSS every 3 minutes — no official Twitter API key required. Falls back through multiple Nitter instances automatically.

📰

Vtuber News Digest

Aggregates posts from r/VirtualYoutubers and r/Hololive every 30 minutes, filters by the vtubers you care about, and optionally generates an AI summary.

🤖

AI Summaries

Powered by OpenAI GPT-3.5-turbo, the optional AI summary condenses up to five news items into 2–3 readable sentences. Privacy-safe — only titles are sent to the LLM.

🛡️

Per-Guild Isolation

Every Discord server manages its own subscriptions independently. Notifications are delivered to the exact channel you configure, with full de-duplication.

⚡

Zero-Friction Setup

Spin up the PostgreSQL database with a single docker compose up command, copy .env.example, fill in your tokens, and the bot is ready.

Bot Commands

Command Description
/track twitch <username> Start tracking a Twitch streamer in this channel.
/track twitter <username> Start tracking a Twitter / X account in this channel.
/track vtuber <name> Subscribe to vtuber news for the given name.
/untrack twitch <username> Stop tracking a Twitch streamer.
/untrack twitter <username> Stop tracking a Twitter / X account.
/untrack vtuber <name> Unsubscribe from vtuber news for the given name.
/list Show all items tracked in this server.

How It Works

  1. 1
    Track — A server member runs a slash command such as /track twitch xqc. The bot stores the guild ID, channel ID, and target username in PostgreSQL.
  2. 2
    Poll — Three independent node-cron jobs poll Twitch (every 60 s), Nitter RSS (every 3 min), and Reddit RSS (every 30 min) in the background.
  3. 3
    De-duplicate — Each new event is checked against a database cache so the same stream, tweet, or article is never posted more than once.
  4. 4
    Notify — The bot builds a rich Discord embed and sends it to the registered channel using channel.send().
